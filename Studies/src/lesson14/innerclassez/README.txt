Внутренние классы
--------------------

Внутренний (вложенный) класс  - класс,
который объявлен внутри класса или интерфейса.

Внутренние классы используются для логической
группировки классов и интерфейсов в одном месте,
чтобы было легче читать и поддерживать.

Внутренний класс может получить доступ ко всем членам
внешнего класса, включая private переменные и методы.

Типы внутренних классов:
- Внутренний класс.
- Локальный внутренний класс.
- Анонимный внутренний класс.
- Статический вложенный класс.

Внутренний класс - нестатический класс, который
создается внутри класса, но вне метода.

Локальный внутренний класс - нестатический класс, который
определяется в методе включающего класса.

Анонимный внутренний класс - класс, который:
- не имеет имени;
- может быть создан только один раз;
- обычно объявляется внутри метода или блока кода,
фигурные скобки заканчиваются точкой с запятой;
- доступен только в том месте, где он определен;
- не имеет конструктора потому, что у него нет имени;
- не может быть статичным.

Статический вложенный класс - статический класс,
созданный внутри класса.
